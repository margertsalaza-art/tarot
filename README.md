# 塔罗占卜 Web 应用

一个基于原生 HTML/CSS/JavaScript 的交互式塔罗牌占卜应用，支持多种经典牌阵，采用粉紫渐变神秘风格设计。

## 在线体验

https://margertsalaza-art.github.io/tarot/

## 核心功能

- 78 张塔罗牌完整数据库，含正位/逆位详细释义
- 多种经典牌阵：单张牌、三张牌、凯尔特十字牌阵
- 流畅交互动画：牌面翻转、洗牌、发牌动画效果
- 响应式布局，完美适配桌面端与移动端
- 神秘粉紫 UI：渐变背景 + 毛玻璃质感卡片

## 技术栈

- 前端：HTML5 + CSS3 + JavaScript (原生)
- 样式：Tailwind CSS (CDN)
- 字体：Google Fonts
- 部署：GitHub Pages
